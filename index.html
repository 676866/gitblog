Everyday Git

What is Git

Git is a tool that helps you save different versions of your work. It is mostly used by people who write code. If you make a mistake, you can go back to an earlier version.

Other tools like Git are SVN, Mercurial, and Perforce. But in this blog, we will only talk about Git and GitHub.

Git Documentation

Installing Git

To check if Git is already on your computer, type this in your terminal:

git --version

To install Git, click the link below:

  • Download Git
  • Then follow the steps for your computer (Windows, Mac, or Linux).

Setting Up Git

Type these commands to tell Git your name and email:

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

Get Started

Create a folder for your work. Inside that folder, make a file named index.html. You can use any code editor like VS Code or any tool you like.

Start Using Git

To see if Git is active in your folder, type:

git status

If it's not active yet, type:

git init

Get Your Files Ready to Save

To tell Git which files to save, type:

git add filename
git add .

Files Git knows are called "tracked". New files are called "untracked" until you add them.

Save Your Work (Commit)

To save a snapshot of your work, type:

git commit -m "Write a short message here"

To see your saved snapshots, type:

git log

To go back to an old version, type:

git checkout [commit-hash]

What are Branches

Branches help you try new ideas without messing up the main work. The main branch is usually called main.

Make a New Branch

git branch newBranch

See All Branches

git branch

The branch with * next to it is the one you are working on.

Switch to Another Branch

git checkout branch-name

Join (Merge) Branches

First go to the branch you want to keep as main:

git checkout main
git merge new-feature

Remove a Branch

git branch -d branch-name
